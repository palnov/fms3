# Product Requirements Document (PRD): Портал "Миграция в Россию"

## 1. Цель проекта
Создать современный, высококачественный сайт для людей (СДЛ), помогающий иностранным гражданам с миграцией в РФ. Проект должен генерировать органический поисковой трафик за счет глубокого экспертного SEO-контента и превосходных поведенческих факторов (ПФ), обеспечиваемых качественным UX/UI и интерактивными инструментами.

## 2. Бизнес-модель
Органический рост за счет SEO и **передача лидов по CPA-модели партнеру (Pravoved/Feedot)**. Основная бизнес-логика сайта заключается в генерации консультационных лидов (имя, телефон, вопрос) через интерактивные формы, плавающие виджеты и интеграцию форм захвата прямо в контент статей.

## 3. Технологический Стек (Vibecoding Approach)
- **Frontend**: Next.js (App Router) / React
- **Стилизация**: Tailwind CSS (или ванильный CSS с современными практиками), Glassmorphism, темная/светлая тема
- **Контентная база**: MDX (Markdown с внедрением React-компонентов)
- **Сборка и деплой**: Vercel / Netlify (SSG/ISR для максимальной SEO-скорости)
- **Отсутствие CMS**: Контент генерируется нейросетью строго по структуре и кладется в репозиторий.

## 4. Целевая аудитория
Иностранные граждане (преимущественно страны СНГ, а также визовые страны), желающие легализоваться в России, работать, учиться, получить РВП, ВНЖ или гражданство. Они часто запутаны в законах, ищут понятные, пошаговые инструкции без "воды".

## 5. Основной функционал (Features)

### 5.1 Интерактивные компоненты (Wow-эффект & ПФ)
- **Migration Path Finder (Калькулятор шансов)**
  - Квиз: вопросы о текущем гражданстве, возрасте, наличии родственников, профессии.
  - Результат: Динамическая дорожная карта (Roadmap) со ссылками на нужные статьи (например: "Вам доступно ВНЖ по профессии, вот инструкция").
- **Генератор чек-листов**
  - Пользователь выбирает свой статус (например, подача на РВП по браку).
  - Генерируется интерактивный To-Do лист со списком документов. Возможность скачать PDF.
- **Тренажер экзамена**
  - Интерактивный UI для прохождения тестов по русскому языку, истории РФ и основам законодательства.

### 5.2 Интеграция AI
- **AI RAG Assistant**: Чат-бот, выступающий в роли цифрового юриста (всплывающий виджет). Обучен только на статьях сайта и актуальной базе ФЗ-115, ФЗ-62. Задача — забирать "хвосты" вопросов и давать прямые ссылки на материалы сайта.
- **AI Content Generator Workflow (Internal Tool)**: Нейросеть (LLM) как движок по написанию статей. ИИ генерирует MDX, вставляя нужные компоненты (`<Alert>`, `<Checklist>`, `<StepByStep>`) по готовой SEO-архитектуре.

## 6. Динамика и Дизайн
- **Aesthetics First**: Дизайн должен быть "премиальным" — никаких шаблонных и дешевых решений. 
- Топовая типографика (Inter/Roboto), гармоничная палитра (возможно, строгая синяя/белая/черная или современные темные интерфейсы).
- Микроанимации при взаимодействии (hover-эффекты, плавное появление элементов).

## 7. Этапы реализации и Статус
*Для подробного отслеживания текущего технического состояния проекта см. файл `docs/implementation_status.md`.*

2. **[X] Базовая Монетизация & Lead Gen**: Интеграция API Правоведа (`/api/leads`), глобальный плавающий виджет `FloatingLawyerWidget`, MDX-баннер `ConsultationBanner`, умная валидация телефона.
3. **[ ] Дизайн-система**: Создание UI-кита (Кнопки, Карточки, Алерты, Аккордеоны для FAQ).
4. **[ ] Core Development**: Интерактивные компоненты (Калькулятор Шансов с переходом в Lead-форму, Генератор Чек-листов).
5. **[ ] Контентная база**: Массовая генерация SILO-структур (статьи про РВП, ВНЖ, Гражданство) с внедренными баннерами консультаций.
6. **[ ] AI Assistant**: Интеграция RAG (Чат-бот, выступающий как первая линия сбора контактов).
7. **[ ] Релиз**: Сборка, тестирование и деплой (Vercel/Netlify), индексация.
