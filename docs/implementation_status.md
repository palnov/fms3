## 1. Архитектура и Базовый Сетап
- [x] Инициализация Next.js (App Router), React 19.
- [x] Настройка Tailwind CSS v4.
- [x] Конфигурация MDX плагинов (`@next/mdx`, `@mdx-js/loader`).

## 2. API & Интеграции
### Партнерская сеть Правовед (Lead Generation)
- [x] **Backend API Route:** Создан `/api/leads` (принимает `name`, `phone`, `question`), проводит валидацию, добавляет `secret` из окружения и проксирует запрос в Feedot CPA.
- [x] **Конфигурация окружения:** `.env.local` и `.env.example` созданы с ключами `PRAVOVED_REFERRAL_ID` и `PRAVOVED_SECRET`.

- [x] **Глобальный виджет (`FloatingLawyerWidget.tsx`):** Липкая (fixed) кнопка "Задать вопрос", открывающая диалоговое окно юриста. Интегрирована глобально в `app/layout.tsx` (отображается на всех страницах).
- [x] **Компонент для статей (`ConsultationBanner.tsx`):** Динамический MDX-совместимый блок, использующий "Мягкий UX" (сначала предлагает нажать кнопку, затем разворачивает форму сбора контактов).
